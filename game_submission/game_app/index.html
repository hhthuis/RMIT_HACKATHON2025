Dòng Lũ Cứu Trợ 🛶

Dùng phím ⬆️⬇️⬅️➡️ hoặc WASD để điều khiển thuyền.

Đón người, né vật cản.

Khi đầy 3 người, lái về một trong hai căn cứ ở góc dưới để thả.

Nếu để trôi 3 người bạn sẽ thua. Thuyền có 3 mạng khi va chạm.

Đã cứu: 0
Sức chứa: 0 / 3
Bỏ lỡ: 0 / 3
Mạng thuyền: 3 ❤️
Nâng (Space): 5 điểm
